h2. Aftalen

FORNAVN indgår i et tæt samarbejde med {{ site.company_name }} fra DATO. Og er derfor en del af teamet i {{ site.company_name }}.

Hvis det efter 2 år vurderes at samarbejdet skal fortsætte tilbydes FORNAVN medejerskab af {{ site.company_legalname }}.

Opgaverne omfatter bl.a. softwareudvikling og rådgivning for {{ site.company_name }}' kunder samt softwareudvikling, systemadministration, markedsføring og salg for {{ site.company_name }}.

Arbejdet udføres dels hos {{ site.company_name }}, dels hos {{ site.company_name }}' kunder.

FORNAVN aftaler med kunderne hvordan de leverede timer kan dokumenteres.

{{ site.company_name }} anbefaler at FORNAVN ikke binder sig til flere timer, end at FORNAVN kan deltage i fælles aktiviteter og i øvrigt har tid til løbende at dygtiggøre sig.

Forslag 1:
Arbejdet skal markedsføres som {{ site.company_name }}-arbejde, og alt arbejde der udføres af NAVN og FIRMANAVN skal faktureres igennem {{ site.company_name }} med mindre andet aftales mellem parterne fra sag til sag.

Forslag 2:
Arbejdet skal normalt markedsføres som {{ site.company_name }}-arbejde, og arbejde der udføres af NAVN og FIRMANAVN faktureres normalt igennem {{ site.company_name }}.
Arbejder der ikke markedsføres som {{ site.company_name }}-arbejde har {{ site.company_name }} intet ansvar for.
